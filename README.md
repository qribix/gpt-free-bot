
# 🧠 GPT Free Bot

Бесплатный Telegram-бот с доступом к ChatGPT и другими моделями через API.

## ⚠️ Проект в архиве

Бот больше не функционирует - сервис Langdock, на котором он был основан, изменил политику использования тестового периода. Код выложен как демонстрация архитектуры и подхода к решению задачи.

## 💡 Идея проекта

Проект показывает, как можно использовать тестовые периоды различных AI-сервисов для организации бесплатного доступа к платным моделям. При грамотной автоматизации можно масштабировать подход на несколько подобных сервисов одновременно - для личного использования или в целях заработка.

Главное, что здесь демонстрируется - не конкретная реализация под Langdock, а сама концепция: ротация ключей, автоматическое переключение при отказах и контроль доступа. Такой подход может быть полезен, если нет возможности развернуть модель локально на своих мощностях или требуется доступ к моделям, которые невозможно запустить на домашнем оборудовании.

## 🚀 Возможности

### Для пользователя
- 🤖 Доступ к ChatGPT и другим моделям (GPT-3, GPT-4, o1 и др.)
- 📝 История сообщений в рамках сессии
- ⚡ VIP-режим - отсутствие задержек между запросами

### Для админа
- 👥 Выдача и отзыв доступа к боту
- ⭐ Управление VIP-статусом пользователей
- 📊 Статистика: общее количество пользователей, с доступом и без
- 🔑 Добавление новых API-ключей в базу
- 🔍 Проверка статуса всех ключей - сколько рабочих из общего числа
- 🔄 Ручной и автоматический поиск рабочего API-ключа
- 📬 Оповещения при ошибках и исчерпании ключей

## 🔧 Как это работало

Бот использовал тестовый период сервиса Langdock: создавалось множество аккаунтов через одноразовые почты, API-ключи заносились в базу данных.

**Система автоматического переключения:**
- При трёх ошибках подряд бот отправлял уведомление админу
- Запускался поиск рабочего ключа среди всех доступных в базе
- Первый рабочий ключ автоматически становился активным
- Если не находилось ни одного - админ получал соответствующее оповещение

**Ограничения:**
- Между запросами обычных пользователей были установлены задержки
- VIP-пользователи работали без ограничений
- Доступ к боту имели только те, кому админ выдал разрешение

## 📦 Установка и запуск

 1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/qribix/gpt-free-bot.git
   cd gpt-free-bot
   ```
   
 2. Установить зависимости:
```bash
pip install -r requirements.txt
```
 3. Открыть `config.py` и вставить свои данные:
> -   `BOT_TOKEN` — токен бота из @BotFather
> -   `ADMIN_ID` — твой Telegram ID
 4. Запустить:
```
start.bat
```
